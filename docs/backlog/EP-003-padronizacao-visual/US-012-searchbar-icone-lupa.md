## User Story
**Como** usuario do sistema,
**Quero** que a barra de pesquisa global tenha um icone de lupa,
**Para** identificar intuitivamente o campo como area de busca.

- [ ] Dado que o icone foi alterado, quando comparo com o padrao de mercado, entao o campo e intuitivo
